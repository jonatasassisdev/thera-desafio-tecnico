# OVGS — Frontend

Interface em Next.js (App Router) + TypeScript + Tailwind CSS. Veja o [README principal do projeto](../README.md) para instruções completas de execução, decisões arquiteturais e documentação.

Resumo rápido:

```bash
npm install
npm run dev   # http://localhost:3002 (ou próxima porta livre)
```

Requer a API backend rodando (padrão: `http://localhost:3333`, configurável via `NEXT_PUBLIC_API_URL`).
