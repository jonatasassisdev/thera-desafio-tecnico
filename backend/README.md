# OVGS — Backend

API REST em NestJS + Prisma + PostgreSQL. Veja o [README principal do projeto](../README.md) para instruções completas de execução, decisões arquiteturais e documentação.

Resumo rápido:

```bash
npm install
npx prisma migrate dev
npm run start:dev   # http://localhost:3333 (Swagger em /docs)
npm test             # testes unitários
npm run test:e2e     # testes de integração
```
